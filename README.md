#  Flexible-Threshold-Model-Stable-ToppLeone

Implementation of a flexible spliced distribution model combining a positively truncated stable tail component with a Topp–Leone exponential body distribution. The repository contains R codes for parameter estimation using (lower) quantile and superquantile, EM optimization, Monte Carlo experiments, and empirical application to informal sector income data.  We  provides various functions for parameter estimation of one-dimensional stable distributions and their mixtures. 

Some of the implemented methods are described in : Solym M. Manou-Abi, Mintodê Nicodème Atchadé, Martin-Junior S. Adechi, and Marc-Hubert Depret (Preprint 2026). Parameter Estimation in a Flexible Threshold Model Combining Positively Truncated Stable and Topp–Leone Exponential Distributions: An Application to Informal Sector Income Data.

The available codes implement the main computational procedures described in the paper, including quantile and superquantile-based initialization, incomplete moment estimation, EM-based maximum likelihood estimation, and optimization algorithms for empirical superquantile estimation.
